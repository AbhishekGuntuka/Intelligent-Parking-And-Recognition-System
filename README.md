# Intelligent-Parking-And-Recognition-System
IPRS — Intelligent Parking &amp; Recognition System. Full-stack IoT solution using ESP32-WROOM-32 &amp; ESP32-CAM with YOLOv8n ANPR, automated servo gates, real-time slot tracking, MQTT/WebSocket communication, Python-Flask backend, SmartCoin wallet, QR token booking &amp; blockchain-linked tamper-proof receipts.
